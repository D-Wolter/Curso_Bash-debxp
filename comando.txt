Não se pergunta se algo da pra fzer no shell, 
a questão é como fazer no shell!


Existe Varios tipos de shell.

cat /etc/shells - lista os shells validos


* sh - bourne shell
* bash - bourne again shell
* rbash - restricted bash
* dash - shell padrão do DSB
* zsh - (z-shell) zero shell
* fish - friendly interactive shell

Comandos Básicos

pwd - mostra pasta atual

ls - mostra itens da pasta

cd nomepasta - entra na pasta

cd/ - vai para raiz do sistema

mkdir nomepasta - cria pasta

rmdir nomepasta - remove diretorio se vazio

help - lista comandos builtin

help comandoBuiltin - esreve help e o nome do comandoBuiltin

ls -la  - mostra todos itens inclusive os .ocultos

ls .bash* - lista tudo que começa com bash

var="banana"
echo $var  - exibe var

echo $0 - indica onde esta sendo executado

> nome - cria arquivo

chmod +x pasta - transforma em exec -> script (cor verde)

nano pasta - abre arquivo no nano editor
cntrl + s -salva
cntrl + x - exit
#!/caminho - primeira linha indicamos caminho do interpretador

(depois de abrir o arquivo script no edito a primeira linha indica o interpretador.)
